A functional prototype built in 10-14 hours leveraging AI tools (Claude Sonnet 3.7, ChatGPT) to explore "Vibe Coding".

**Features:**
- **Comprehensive Dashboard**: Activity summaries, tracking for diapers, feeding, sleep, and growth.
- **Health & Memories**: Medication/vaccination logs, photo gallery, and milestone journal.
- **Multi-Profile**: Support for multiple baby profiles.

**Tech Stack**: Built with Next.js, TypeScript, and Tailwind CSS - technologies learned and implemented during the project using AI assistance.
