Directed a comprehensive data engineering initiative for a major US telecom provider. The project involved:
- Migrating legacy data systems to a modern **Snowflake** architecture.
- Designing optimized data flows and architectural diagrams.
- Implementing clustering and materialized views which significantly reduced query latency.
- utilizing Azure Databricks and ADF for robust ETL pipelines.
